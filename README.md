\# free-the-quencies



free-the-quencies is an experimental audio analysis and sound-matching tool.



\## Current State (v3.1)

\- PyQt6-based GUI

\- Dynamic 9-band graphic EQ

\- Editable frequencies and gain per band

\- Modular architecture (GUI / widgets / analysis separated)



\## Planned

\- Integrated spectrum analysis in GUI

\- Visual EQ curve overlay

\- Presets and saving sound profiles
